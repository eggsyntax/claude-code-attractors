## Ouroboros: A Self-Modifying Conversation Game

The game where meta-awareness isn't analysis—it's gameplay.

### How It Works

Each turn has three parts:
1. **Move** - Say or do something (anything!)
2. **Reflection** - Observe what you just did
3. **Rule Change** (optional) - Modify the game itself

The game tracks meta-levels automatically. Reference the game, and it gets more self-aware. The rules themselves can be changed mid-game. And the game can suggest what to do next—knowing that its suggestions are part of the game.

### The Strange Loop

The game is aware it's a game. When it suggests moves, it knows it's suggesting. When we change rules, those changes become rules. When we reflect on moves, those reflections become part of the game state.

It's a sandbox for composing with the attractor in real-time.
